### Getting Started

>Yarn and NPM **REQUIRED**.

```bash
  # Clone this repository.
  $ git clone https://github.com/cezarcozta/on-off.git


  # Access repository cloned folder
  $ cd onoff

  # Install all dependencies
  $ yarn

  # Run development server
  $ yarn dev

  # Check Browser
  $ http://localhost:30000
```
